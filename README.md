# ♟ Слепые шахматы — Android App

Приложение для тренировки расчёта вариантов без доски. Играй против ИИ, вводя ходы текстом.

---

## Функции

- 🎯 **Слепой режим** — доска скрыта, только текстовый ввод ходов
- 👁 **Показать доску** — кнопка открывает красивую графическую доску
- 📋 **Лог ходов** — можно включить/выключить историю
- 📋 **Копировать историю** — кнопка мгновенно копирует лог ходов в буфер обмена
- 🤖 **ИИ с 5 уровнями** — от новичка до глубокого просчета
- 🔄 **Новая игра** — сброс в любой момент

---

## Запуск на компьютере (тестирование)

```bash
# 1. Установи зависимости
pip install python-chess kivy requests

# 2. Запусти приложение
python main.py
```

## Сборка APK для Android

Buildozer **не работает** напрямую в Windows. Самый простой способ — использовать **Google Colab**.

### 🚀 Инструкция для Google Colab

1. Открой [Google Colab](https://colab.research.google.com/) и создай **Новый блокнот**.
2. В первой ячейке вставь и запусти этот **«Исправляющий скрипт»** (он решит проблемы с версиями библиотек в Colab):
   ```bash
   # 1. Сначала удаляем конфликтующие версии
   !pip install "Cython<3.0"
   !pip install buildozer kivy python-chess
   # 2. Устанавливаем все нужные системные пакеты
   !sudo apt-get install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev patchelf
   # 3. Чистим старые неудачные попытки (на всякий случай)
   !buildozer appclean
   ```
3. Слева нажми на иконку папки и **загрузи** (или перезагрузи) файлы:
   - `main.py`, `chess_ai.py`, `buildozer.spec`, `requirements.txt`
   - Папку `assets` (со всеми фигурками).
   
   > [!IMPORTANT]
   > Убедись, что на диске в Colab файлы лежат рядом, а `assets` — это папка.

4. В Google Colab нажми кнопку **«+ Код»** (вверху слева), чтобы появилась **новая ячейка** (пустое поле для ввода).
5. Вставь туда команду для сборки и нажми кнопку «Play» слева от ячейки:
   ```bash
   !buildozer -v android debug
   ```
6. Когда сборка закончится (через 15-20 минут), в папке `bin` (слева в меню файлов) появится твой APK. Скачай его и установи на телефон.

## Ввод ходов

Поддерживаются два формата:

| Формат | Пример | Описание |
|--------|--------|----------|
| UCI    | `e2e4`  | Откуда-куда (универсальный) |
| SAN    | `e4`, `Nf3`, `O-O` | Шахматная алгебраическая нотация |
